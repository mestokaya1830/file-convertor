# DataConverter UI | Vue 3 Reactive Interface

The modern, reactive frontend for the DataConverter API, built with Vue 3 to provide a seamless file upload and data transformation experience.

## 🚀 Key Features
* **Reactive Data Handling:** Leveraged Vue 3's **Composition API** for efficient state management and real-time UI updates.
* **Smart File Uploads:** Integrated drag-and-drop support with immediate client-side validation before hitting the API.
* **Dynamic Feedback:** Real-time progress tracking and automated error handling for file size and type mismatches.
* **Component-Based Architecture:** Modular design ensuring high reusability and maintainable code structure.

## 🛠 Technical Implementation
* **Composition API (script setup):** Utilized `ref`, `reactive`, and `computed` properties for clean and scalable logic.
* **Axios Integration:** Orchestrated asynchronous API calls to the Node.js backend with custom request/response interceptors.
* **Conditional Rendering:** Implemented seamless transitions between upload, processing, and download states.
* **Vite Powered:** Built on **Vite** for lightning-fast development HMR and optimized production bundles.

## 💻 Tech Stack
* **Framework:** Vue 3 (Composition API)
* **Build Tool:** Vite
* **HTTP Client:** Axios
* **Styling:** CSS3 (Modern Flexbox/Grid)
